## 一句话结论

## 二、Donchian 突破 vs Faber 均值（同 timeframe 对照实验）

两策略都跑月线 + equal-weight + combined-27，**唯一差异是 Decider 算法**：
- **Faber**：close > MA10（10 月均值）→ BUY；close ≤ MA10 → SELL（均值穿越）
- **Donchian**：close > max(过去 10 月 close) → BUY；close < min(过去 5 月 close) → SELL（极值突破）

| 维度 | Faber 均值法 | Donchian 极值法 |
|---|---|---|
| 入场频率 | 较高（均值穿越易触发） | 较低（必须创 N 月新高） |
| 出场速度 | 较快（close ≤ MA 立即翻 DOWN） | 较慢（必须破 N/2 月新低） |
| 4 窗口 ΔCAGR vs baseline | 3年 -2.63 / 5年 -3.11 / 8年 **+1.00** / 10年 **+6.15** | 3年 -5.05 / 5年 -3.00 / 8年 -2.45 / 10年 -14.25 |
| 4 窗口 ΔMaxDD vs baseline | 3年 -3.36 / 5年 -5.83 / 8年 -2.92 / 10年 **+3.03** | 3年 -3.30 / 5年 **-0.66** / 8年 **+1.05** / 10年 **+15.97** |

**核心结论**：
- 均值法（Faber）：**追长期 CAGR 的赢家**。8-10y 窗口反超 baseline，是 cycle 1 修 bug 后最大反转。
- 极值法（Donchian）：**追低回撤的赢家**。所有 5y+ 窗口 MDD 都比 baseline 浅，10y MDD 比 baseline 少回撤 16pp，但 CAGR 也付出 14pp 代价。

两者非此即彼——选哪个看投资目标：长期 CAGR 选 Faber，低回撤选 Donchian。

之前 v9.3-bear "加 Filter 减信号"路线已被 cycle 1 / hotfix 双重确认是错路（4 窗口全输 -0.06 ~ -2.31pp）。本周期对比再次验证：**"换 Decider 算法"路线（Faber / Donchian）有效，"加 Filter"路线无效**。

### 分指数模式

按 universe 子集看 donchian-200 vs baseline（参考上方 Δ (donchian-200 − v9-baseline) 子表）：

- **A 股宽基**（沪深300/上证50/中证500/中证1000/中证2000/创业板50/科创50/北证50）：主流宽基（沪深300 +1.48 / 上证50 +1.07 / 中证500 +4.25 / 创业板50 +3.82）CAGR 微赢 baseline，**但 MDD 普遍加深 6~15pp**（沪深300 +13.37 / 中证500 +14.08 / 创业板50 +15.42）——突破法在 A 股大盘上拿 CAGR 但抗跌反不如 baseline。北证50 -45.90pp CAGR 是单点灾难（小盘流动性差，10/5 月极值穿越频繁假信号）。
- **A 股主题/行业**（光伏/白酒/医疗/5G/AI/智汽车/军工/新能/有色/化工 等）：**几乎全员 CAGR 输 baseline -0.49 ~ -10.02pp**（5G -10.02 / 白酒 -8.41 / 医疗 -7.31 / 智汽车 -7.02 / 军工 -3.85 等），但 **MDD 多数比 baseline 浅** 5~25pp（5G -25.36 / 白酒 -14.98 / 智汽车 -14.81 / 军工 -14.90 / 光伏 -13.54）——主题行业波动大、噪声多，Donchian "难入场"反成抗跌优势。新能车 +7.44/+19.42 是反例（CAGR 大赢但 MDD 也深）。
- **港股**（HSI/HSCEI/HSTECH）：HSI +4.41 / HSCEI +3.80 CAGR 显著赢 baseline，**但 MDD 都加深 11~12pp**（吃下 2018-2022 港股大熊整段下跌），HSTECH 反向 -12.55 CAGR / -14.03 MDD（恒科上市仅几年，月线突破信号样本不足）。
- **美股**（NDX/SPX）：CAGR 接近 baseline（-0.51 / -1.29），MDD 都比 baseline 浅（-6.73 / -2.26）——**美股月线噪声低，Donchian 的稳健画像在这里最纯粹**（CAGR 持平 + MDD 更浅）。
- **加密 / 商品**（BTC/XAU/XAG）：BTC **-34.16pp CAGR 是 universe 内最大单点拖累**（与 Faber 的 -34.74pp 几乎一致——所有月线策略对 BTC 都束手无策，过去 cycle 报告反复印证），但 MDD 微浅 -5.55pp。黄金 +0.48/-6.09（CAGR 持平 + MDD 更浅，是 Donchian 适配标的的范本）；白银 +3.86 CAGR 但 MDD 加深 8.65。

总结：donchian-200 在**美股、黄金、A 股主题行业**上的"低 CAGR + 低 MDD"防御画像最清晰，**A 股宽基/港股大盘**则反而是"赢 CAGR 输 MDD"——突破法在不同市场的表现并不一致，组合层面的"5y+ MDD 全部更浅"是各子集互相抵消后的净效果。BTC 永远是月线策略的单点黑洞，建议未来组合层把 BTC 单独剔除或换日线策略。

## 五、后续方向

### 当前 4 策略实操判断

| 策略 | 适合谁 | 理由 |
|---|---|---|
| v9-baseline | 平衡型投资者 | 总收益最高、所有窗口 CAGR 居中或居首 |
| **faber-gtaa** | **长期投资者（8+ 年视野）** | 8-10y 大幅跑赢 baseline；CAGR + 回撤双赢 |
| **donchian-200** | **风险厌恶投资者** | 牺牲 CAGR 换 MDD，10y 少回撤 16pp |
| v9.3-bear | 不推荐 | 4 窗口全输；"加 Filter"路线确认无效 |

### 已弃 + 已得 + 待办

**已弃**（明确不再投入）：
- v9.3-bear 路线（加 BearTrendFilter）：cycle 1 + cycle 2 双重确认全输

**已得**（有明确价值的策略）：
- v9-baseline：通用基准，长期不需替换
- faber-gtaa：长期投资者升级方案
- donchian-200：低 MDD 防御方案

**待办**（cycle 3+）：
1. **下一周期 A：Dual Momentum**（横截面 top-K 选股）：与 Faber/Donchian 的"全 universe in/out"逻辑互补，预计在主题/行业子集上有 alpha 空间
2. **混合策略**：baseline + Faber 各 50% 的组合配置，需要新 portfolio aggregator
3. **参数扫描**：Faber MA8 / MA12 + Donchian 8/4 / 12/6 等变体，注册即跑

### 调参方向

按代价从小到大：
1. **改窗口**：注册新策略名（如 `faber-gtaa-ma12` / `donchian-12-6`）即跑，框架支持
2. **混合 universe**：仅在大盘宽基/港股 子集应用 Faber，主题用 baseline。需要 universe-aware 配置
3. **加 stop-loss**：Faber/Donchian 原版无止损。可加"持仓回撤超 N% 强制清仓"
4. **位置规模管理**：海龟系统经典扩展是按 ATR 决定 position size。需要新 aggregator
